  Проект состоит из четырёх модулей и файла формата json где хранятся все данные о книгах.
В модуле «main.py» содержится точка входа в программу через класс «Menu». Этот класс также
содержит главное меню библиотеки, через которое осуществляется выбор подменю действий с
книгами. Модуль «command_line_sub_menu.py» через класс «SubMenu» предоставляет набор подменю,
которые поддерживают ввод пользователем данных, работу с модулем базы хранения книг и вывод данных
с базы обратно пользователю. Модуль «books_data_library_control.py» представляет собой систему
управления базой данных с соответствующим набором функций. Модуль «custom_string.py» предоставляет
класс «CustomString», который позволяет добавлять escape-коды ANSI к строкам в зависимости от типа
сообщения. Это позволяет изменять цвет, цвет фона и стиль выводимых строк.
